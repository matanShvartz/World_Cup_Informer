World Cup Informer:

- Server-Client program to publish and get updates about the world cup
- Events are published by clients to other clients
- Using TCP protocol
- Client is written in C++
- Server is written in Java
- Server supports both Thread-Per-Client and Reactor design patterns, based on input upon startup
- This project was developed as part of a university course at Ben Gurion University - SPL course
- The emphasis was on interprocess communication using network protocols over sockets

Enjoy!
